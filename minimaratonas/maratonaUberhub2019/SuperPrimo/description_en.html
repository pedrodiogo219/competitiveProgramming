Super Primes

By Pedro Diogo Machado, Universidade Federal de Uberlândia Brazil

Timelimit: 1

Prime numbers are the natural numbers that have exactly two different divisors: the number 1 and itself. For example, the numbers 2, 3 and 5 are primes. In the other hand the number 12 is not prime as he has 6 divisors: 1, 2, 3, 4, 6 and 12. The number 1 is not prime because it has only one divisor.

    The prime numbers are really important for the mathematics, but something a lot of people forget is that the Super Primes exist. The Super Primes are those that have exactly three divisors. For example, the number 4 is a Super Prime, because its divisors are 1, 2 and 4.

    Your task is to look at a list of number and tell wich of them are Super Primes.

 

Input

You will receive an integer N indicating the number of numbers in the list.

Follow N lines, each one containing one integer Aᵢ

0 < N <= 10³

1 <= Ai <= 10⁶

Output

For each value Aᵢ, print one line contain only the character ‘S’ if Aᵢ is a Super Prime. Otherwise, print the character ‘N’.
